개인정보 활용 동의서



본 웹사이트는
2020 POSTECH UGRP(학부생 연구 프로그램)의 결과로써
딥러닝 모델 VGG로 추출한 얼굴의 feature 값을
ML(machine learning) 기법으로 학습한 모델로
MBTI를 예측하는 목적으로 제작되었습니다.

본 웹사이트는 향후 수집된 얼굴들로
모델의 성능을 더욱 향상시키고자 합니다.
아래 약관에 동의하신다면,
입력된 얼굴은 향후 모델을 다시 학습하는데 사용됩니다.
수집된 얼굴은 "모델 학습" 외 다른 목적에
전혀 사용되지 않습니다.

약관에 동의하지 않으신다면,
VGG로 추출한 얼굴 feature 값만을 저장해서
MBTI 값을 에측하는 데에만 사용됩니다.


아래 약관에 동의하시겠습니까?
